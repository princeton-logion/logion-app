# χαῖρε!
Logion is an open-source, no-code application designed to assist philologists using state-of-the-art language models.
With Logion, researchers can download a suite of pre-modern Greek langauge models for conducting
philological research tasks, such as error word detection and gap filling. Logion is designed to be run on 
personal computers without access to high performance computing (HPC) resources.

## Installation
Logion is available for Windows and MacOS. We hope to make the application publicly available on our website soon.

For the time being, only researchers who have been provided a Google Drive link can download the Logion beta application. To do so, click the version of Logion corresponding to your device's operating system ("logion-Win" for Windows, "logion-mac" for MacOS). 

After the .zip file has downloaded, double-click the .zip file in your Downloads folder. Open the resulting folder, and double-click the "Logion" application. This will open and install Logion on our system.

Note that, if your institution restricts downloads, you may need to contact your IT administrator to enable downloading new programs temporarily on your device.

## Requirements
Logion requires no outside software or installations.
