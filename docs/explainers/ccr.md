# Chance-confidence ratio

Logion's error detection algrothim utilizes what is called the chance-confidence ratio. This is an original algorithm developed by Logion team members and first introduced in this [ACL paper](https://aclanthology.org/2023.alp-1.20/). The alogirthm determines the likelihood that a given word was mistranscribed at a certain point in the textual history.'
