# περῐ́ τῆς μηχᾰνῆς

Hardware affects software. Logion does not require access to high-performance computing (HPC) resources. It is designed to run on local machines with divers hardware and operating systems. But although the specific hardware on which Logion runs does not affect the accuracy or quality of its predictions, hardware will affect the speed at which Logion performs its operations. Thus, for perhaps the first time in history, philologists may want to be aware of their computer's hardware. The following provides a brief overview and guide to help users navigate hardware questions specifically in relation to Logion.

## Which hardware is best?

Hardware will not affect the accuracy or quality of predictions generated by Logion models. But hardware *will* affect the speed at which Logion generates predictions.

### CPU vs. GPU

Nearly every computer nowadays has both a central processing unit (CPU) and graphics processing unit (GPU). Both are processors that use memory to store/access data, perform calculations, and execute instructions. In short, these constitute the computer's so-called brain. CPUs are responsible for general-purpose computing tasks (word processors, web browsing, etc.); GPUs are responsible for computationally expensive, high-throughput tasks (video games, training AI models, etc.).

When you start Logion, it detects which type of GPU is available on your machine. If your machine has a compatible GPU, it utilizes that GPU to generate results. If not, it defaults to your machine's CPU, which is slower. Our team's internal experiments suggest that, when executing Logion's error detection task on a sequence of 122 words in Byzantine Greek, a GPU speeds up performance by approximately 40% compared to the same machine's CPU. Logion is compatible with the following types of GPUs:

- Nvidia
- Apple Silicon (M-chip)
- Intel Graphics

## How do I find which GPU my computer has?

#### On macOS:
1. Navigate to the top-left corner of your computer's menu bar.
2. Click the Apple Logo ().
3. Select 'About This Mac" from the drop-down menu.
4. Your computer's GPU should appear beside "Graphics" or "Chip".

#### On Windows:
1. Right-click on the taskbar.
2. Select 'Task Manager'.
3. Click the 'Performance' tab in the Task Manager window.
4. Scroll down and select 'GPU 0'.
5. GPU information for your computer should appear in the top-right corner.
